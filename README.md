## 技术栈
vue3+vue-router+webpack+ES6+vue-axios+express+mysql


## 已完成功能
```
登录注册
根据菜谱名搜索菜谱
发布菜谱
修改菜谱
删除菜谱
收藏菜谱
取消收藏菜谱
发布菜谱评论
删除菜谱评论
根据当前季节推送当季菜谱
```

## 项目运行
```
git clone https://github.com/SprinGreat/menu-01.git


将menu-01文件夹下的test.sql数据库文件导入数据库


cd menu-01


npm install


运行server文件夹下的express.js文件，开启服务器


npm run serve
```
